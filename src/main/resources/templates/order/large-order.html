주문하기

대량 주문은 양식 다운로드를 하신 뒤, 엑셀 파일에 주문 내용을 적은 후 엑셀 파일을 업로드 하면 자동으로 주문이 추가됩니다.
추가된 내용 확인 후 제출하기 버튼을 누르면 주문이 완료됩니다.

보내는 사람, 전화번호 : 보내는 사람을 변경하고 싶으실 시, 정보를 변경해주시고, 아니라면 , 번호로 입력하시면 됩니다.
받는 사람, 전화번호 : 수령할 사람의 이름, 전화번호로 적으시면 됩니다.
우편번호 : 우편번호를 적어주시면 됩니다.
주소 : 주소를 적어주시면 됩니다.
상세주소 : 동, 호수 등 상세 주소를 적어주시면 됩니다.
품종 : 현재 판매중인 품종을 정확히 적어주시면 됩니다. (홍로, 부사, 시나노골드 등)
규격 : 주문하시고 싶은 사과의 호수를 단위 없이 적어주시면 됩니다.(1번과 주문시 1, 특 주문시 특)
키로수 : 5, 10키로 중 주문하시고 싶은 사과의 키로수를 단위 없이 적어주시면 됩니다.(5kg 주문시 5, 10kg 주문시 10)
수량 : 주문하시고 싶은 사과의 수량을 단위 없이 적어주시면 됩니다.(1박스 주문시 1, 5박스 주문시 5)
표를 추가하시면 추가하시는 줄 개수대로 주문이 늘어나게 됩니다. 정확하게 확인 후 등록해주시기 바랍니다.
이름을 입력해주세요
전화번호를 입력해주세요
비밀번호(숫자6자리)를 입력해주세요
